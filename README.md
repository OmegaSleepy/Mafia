# WIP

Mafia (title work in progress) is an adaptation of the [social game mafia](https://en.wikipedia.org/wiki/Mafia_(party_game)). This is a Minecraft mod for 1.21.1 Neoforge, no there won't be a Fabric port.   
The project (will) use owo-lib for nice looking ui and integrate simple voice chat for dinamic muting and other social features.
